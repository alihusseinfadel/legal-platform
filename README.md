## 📖 نظرة عامة

**منصة القانون الإداري** هي منصة ذكية متكاملة للاستشارات القانونية الإدارية العراقية، تجمع بين قاعدة بيانات شاملة للقوانين الإدارية العراقية والذكاء الاصطناعي لتقديم تحليل قانوني دقيق وسريع.

## ✨ الميزات

### 💬 المستشار القانوني الذكي
- دردشة تفاعلية بالعربية
- Streaming للردود (عرض تدريجي مثل ChatGPT)
- **محادثات متعددة** مع حفظ تلقائي
- تصدير المحادثة كملف **PDF** احترافي
- يعتمد على موديل GPT-OSS 120B عبر OpenRouter

### 📄 تحليل الوثائق
- رفع ملفات **PDF, Word, صور, نصوص**
- تحليل قانوني فوري مع تحديد السند القانوني
- توصيات قانونية مفصلة

### 📚 قاعدة القوانين
- **9 قوانين** إدارية عراقية شاملة
- **31 مادة قانونية** كاملة
- **بحث ذكي** مع ترتيب حسب الصلة
- تطبيع النص العربي (تشكيل، ألف، هاء)

### 📖 قاموس المصطلحات
- **29 مصطلح** قانوني مع تعريف كامل
- السند القانوني لكل مصطلح
- فلترة وبحث متقدم

### ⚖️ دراسات الحالة
- **6 قضايا واقعية** مع التحليل
- السيناريو + التحليل + السند + الخلاصة

### 📋 إدارة الاستشارات
- تسجيل ومتابعة الاستشارات
- إحصائيات شاملة
- حالات متعددة (جديدة، قيد المراجعة، مكتملة، مغلقة)

### 🎨 واجهة احترافية
- تصميم بألوان وزارة التعليم العالي
- **خلفية مائية** بشعار جامعة ديالى
- **شريط جانبي** احترافي بألوان داكنة
- **ترحيب صوتي تلقائي** للمستخدم
- تخصيص حجم الخط
- دعم RTL كامل

## 🚀 التثبيت

### المتطلبات
- Python 3.10 أو أحدث
- pip

### خطوات التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/alihusseinfadel/legal-platform.git
cd legal-platform

# إنشاء بيئة افتراضية
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# تثبيت المكتبات
pip install -r requirements.txt

# نسخ ملف الأسرار
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
# ثم حرر الملف وأدخل مفتاح OpenRouter الخاص بك
```

### الحصول على مفتاح OpenRouter API (مجاني)
1. اذهب إلى [openrouter.ai](https://openrouter.ai)
2. سجّل دخول (Google/GitHub)
3. اذهب إلى [openrouter.ai/keys](https://openrouter.ai/keys)
4. اضغط "Create Key"
5. انسخ المفتاح وضعه في `.streamlit/secrets.toml`

## 💻 الاستخدام

```bash
streamlit run app.py
```

ثم افتح المتصفح على: `http://localhost:8501`

## 🛠️ التقنيات المستخدمة

- **Streamlit** - إطار عمل الويب
- **OpenRouter AI** - واجهة الذكاء الاصطناعي
- **ReportLab** - توليد ملفات PDF
- **arabic-reshaper + python-bidi** - معالجة النص العربي
- **Leaflet** - الخرائط التفاعلية (في لوحة الأدمن)
- **Google Translate TTS** - الترحيب الصوتي

## 🔐 الأمان والخصوصية

- لا يتم حفظ أي بيانات شخصية
- جميع المحادثات تُخزّن محلياً فقط
- مفتاح API محمي في ملف `secrets.toml` غير مرفوع
- كل الاتصالات عبر HTTPS
